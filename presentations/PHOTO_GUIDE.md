# Инструкция по добавлению фотографий в презентацию

## 📷 Какие фотографии нужны и где их разместить

### Слайд 6: Специализированное оборудование

**Требуется 3-4 фото:**

1. **Токарные станки с ЧПУ**
   - Покажите современное оборудование
   - Желательно в процессе работы
   - Размер: ~400×300px каждое

2. **Фрезерные станки**
   - Крупногабаритный фрезерный станок
   - Показать размер (1200×5000 мм)
   - Размер: ~400×300px

3. **Шлифовальные станки**
   - Круглошлифовальный или плоскошлифовальный
   - В рабочем состоянии
   - Размер: ~400×300px

**Размещение:**
- Создайте сетку 2×2 в нижней части слайда
- Или 3 фото в ряд
- Добавьте подписи под каждым фото

**В PowerPoint:**
```
Вставка → Рисунки → выберите файлы
Расположите в сетке
Формат → Эффекты → Тень (опционально)
```

### Слайд 7: Грузоподъемное оборудование ⭐ ВАЖНО!

**КРИТИЧЕСКИ ВАЖНО: Фото мостового крана 160 тонн!**

Это **УНИКАЛЬНОЕ** преимущество компании!

**Требуется:**

1. **Мостовой кран 160 тонн (Воронеж)** 
   - Главное фото - крупным планом
   - Покажите масштаб крана
   - Желательно с людьми для сравнения
   - Размер: большое фото ~800×600px

2. **Производственный цех с кранами**
   - Общий вид производства
   - Видно грузоподъемное оборудование
   - Размер: ~600×400px

3. **Процесс работы крана** (опционально)
   - Кран поднимает крупную деталь
   - Демонстрация возможностей
   - Размер: ~600×400px

**Размещение:**
- Основное фото крана 160т - в центре слайда (крупно!)
- Дополнительные фото - меньше, сбоку или снизу
- Добавьте **яркую надпись**: "МОСТОВОЙ КРАН 160 ТОНН!"

**Текст на слайде:**
```
⭐ УНИКАЛЬНОЕ ПРЕИМУЩЕСТВО ⭐
МОСТОВОЙ КРАН 160 ТОНН
Обработка деталей до 150 тонн
```

### Слайд 8: Наша команда (опционально)

**Требуется 1-2 фото:**

1. **Производственная команда**
   - Группа специалистов
   - В рабочей одежде/спецодежде

   - Размер: ~600×400px

- Не перегружайте - текст важнее

### Слайд 13: Портфолио работ ⭐ ВАЖНО!

**Требуется 4-6 фото готовых работ:**

1. **Крупногабаритные валы**
   - После токарной обработки
   - Покажите размер и качество
   - Размер: ~400×300px

2. **Зубчатые колеса/шестерни**
   - Результат работы участка изготовления шестерен
   - Крупный план для демонстрации точности
   - Размер: ~400×300px

3. **Револьверные головки**
   - Готовая продукция
   - Уникальный товар компании
   - Размер: ~400×300px

4. **Сборочные работы**
   - Процесс сборки станка
   - Работа сборочного участка
   - Размер: ~400×300px

5. **До/После ремонта** (опционально)
   - Сравнение станка до и после модернизации
   - Наглядная демонстрация качества работы
   - Размер: 2 фото по ~400×300px

6. **Крупный проект**
   - Фото особо значимого проекта
   - Для Севмаша или Звездочки (если есть)

- Создайте галерею 2×2 или 3×2

- Добавьте краткие подписи под каждым
- Оставьте место для основного текста вверху

## 🎨 Требования к фотографиям

### Технические требования:

**Формат:**
- JPG или PNG
- Разрешение: минимум 1024×768px
- Размер файла: до 2MB каждое

**Качество:**
- Четкие, в фокусе
- Хорошее освещение
- Нет размытия
- Профессиональный вид

**Содержание:**
- Чистое оборудование (протрите перед съемкой!)
- Без посторонних предметов в кадре
- Показывайте масштаб
- Лучше несколько ракурсов

### Обработка фотографий:

**Базовая обработка (рекомендуется):**
1. Обрезка - уберите лишнее по краям
2. Яркость/контраст - сделайте фото четче
3. Цветокоррекция - естественные цвета
4. Уменьшение размера - до 1920×1080px максимум

**Инструменты:**
- Windows: Встроенный редактор "Фотографии"
- Онлайн: Canva.com, Photopea.com
- Профессионально: Photoshop, GIMP

## 🚀 Быстрый способ добавления фото

### В PowerPoint:

1. **Откройте нужный слайд**

2. **Вставка → Рисунки → Это устройство**

3. **Выберите все нужные фото сразу** (Ctrl+клик)

4. **Расположите фото:**
   - Выделите все фото (Ctrl+A)
   - Формат → Выровнять → Распределить по горизонтали
   - Формат → Выровнять → Распределить по вертикали

5. **Примените эффекты:**
   - Выделите все фото
   - Формат → Эффекты рисунка → Тень (Смещение вниз)
   - Или: Формат → Граница рисунка → 2pt белая рамка

6. **Сгруппируйте** (Ctrl+G) для удобства перемещения
