# Инструкция по конвертации презентации в PowerPoint

## 📋 Способы конвертации Markdown → PowerPoint

### Способ 1: Ручная конвертация (Рекомендуется для лучшего результата)

#### Шаг 1: Откройте PowerPoint
- Создайте новую презентацию
- Выберите темную тему (профессиональный вид)

#### Шаг 2: Настройте мастер-слайды
**Цветовая схема:**
- Фон: Темный (#1a1a2e или #16213e)
- Акцентный цвет: Голубой (#38bdf8)
- Текст: Белый (#ffffff)
- Дополнительный: Серый (#94a3b8)

**Шрифты:**
- Заголовки: Arial Bold, 32-44pt
- Подзаголовки: Arial, 24-28pt
- Основной текст: Arial, 18-20pt

#### Шаг 3: Создайте слайды по структуре

**Слайд 1 - Титульный:**
```
Заголовок: ⚙️ ТД РУССтанкоСбыт
Подзаголовок: Ремонт, модернизация и производство станочного оборудования
              🚢 Комплексные решения для судостроительной промышленности

Текстовый блок:
🏆 Надежный партнер с 2014 года
12 лет опыта | 4 площадки | 15 000+ м² | 60+ специалистов

Контакты внизу:
📧 zakaz@tdrusstankosbyt.ru
📱 +7 (499) 390-85-04
🌐 tdrusstankosbyt.ru
💬 t.me/tdrusstankosbyt
```

**Слайд 2 - О компании:**
- Используйте таблицу 2x2
- Левая часть: "РЕМОНТ И МОДЕРНИЗАЦИЯ" с маркерами
- Правая часть: "ПРОИЗВОДСТВО И ПОСТАВКА" с маркерами
- Внизу: рамка с текстом про клиентов

**Слайд 3 - Ключевые показатели:**
- Создайте 6 текстовых блоков в 2 ряда
- Каждый блок содержит крупную цифру и пояснение
- Примеры:
  - 🗓️ **12 ЛЕТ** на рынке
  - 🏭 **4 ПЛОЩАДКИ** производственных

#### Шаг 4: Добавьте визуальные элементы

**Для блоков с рамками:**
- Вставка → Фигуры → Прямоугольник
- Настройте границу и заливку
- Добавьте текст внутрь

**Для таблиц:**
- Вставка → Таблица
- Настройте цвета ячеек (bgcolor из Markdown)

**Для иконок:**
- Используйте эмодзи из Markdown как есть
- Или замените на профессиональные иконки (Вставка → Значки)

### Способ 2: Автоматическая конвертация с помощью Pandoc

#### Установка Pandoc:
**Windows:**
1. Скачайте с https://pandoc.org/installing.html
2. Установите программу
3. Откройте командную строку

**Использование:**
```bash
# Базовая конвертация
pandoc Prezentacija_Zvezdochka.md -o presentation.pptx

# С настройками
pandoc Prezentacija_Zvezdochka.md -o presentation.pptx --reference-doc=template.pptx
```

**Примечание:** Автоматическая конвертация может потерять форматирование ASCII-арт блоков.

### Способ 3: Онлайн-конвертеры

**Рекомендуемые сервисы:**
1. **Marp** (https://marp.app/)
   - Специально для презентаций из Markdown

2. **Slides.com**
   - Импорт Markdown
   - Онлайн-редактор
   - Экспорт в PowerPoint

3. **CloudConvert** (https://cloudconvert.com/md-to-pptx)
   - Простая конвертация
   - Без регистрации

## 🎨 Рекомендации по адаптации

### 1. Замените ASCII-арт на визуальные элементы PowerPoint

**Было в Markdown:**
```
╔════════════════════════════╗
║ МОСТОВОЙ КРАН 160 ТОНН! 🌟 ║
╚════════════════════════════╝
```

**В PowerPoint:**
- Вставьте прямоугольник с закругленными углами
- Заливка: Градиент голубого
- Границ: Толстая линия (#38bdf8)
- Текст: "МОСТОВОЙ КРАН 160 ТОНН! 🌟"

### 2. Используйте цветовые акценты

**Цвета для блоков (из Markdown):**
- `bgcolor="#e8f5e9"` → Светло-зеленый (Качество)
- `bgcolor="#e3f2fd"` → Светло-голубой (Оперативность)
- `bgcolor="#fff3e0"` → Светло-оранжевый (Надежность)

### 3. Выделите ключевые цифры

**Используйте:**
- Размер шрифта 48-60pt для цифр
- Жирный шрифт
- Контрастный цвет (#38bdf8)

### 4. Добавьте фотографии

**Рекомендуемые места:**
- Слайд 6: Фото специализированного оборудования
- Слайд 7: Фото мостового крана 160 тонн
- Слайд 13: Фото выполненных работ (галерея 2x2)

## ⚡ Быстрый старт (15 минут)

1. **Откройте PowerPoint** → Новая презентация
2. **Выберите тему**: Темная (Ion, Berlin, или Frame)
3. **Создайте 16 слайдов** (Главная → Создать слайд)
4. **Откройте файл Prezentacija_Zvezdochka.md** в браузере/текстовом редакторе
5. **Копируйте содержимое** каждого слайда:
   - Заголовки → в заголовки PowerPoint
   - Списки → в текстовые блоки
   - Таблицы → создайте таблицы в PowerPoint
6. **Добавьте визуальные элементы**:
   - Фигуры для рамок
   - Цветные блоки для преимуществ
7. **Сохраните** в формате PPTX
