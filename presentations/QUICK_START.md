# Быстрая генерация PowerPoint презентации

## 🚀 Автоматическая генерация через Python

Я создал Python скрипт для автоматической генерации PowerPoint файла!

### Метод 1: Использование Python скрипта (БЫСТРО - 1 минута)

#### Шаг 1: Установите Python-библиотеку
```bash
pip install python-pptx
```

#### Шаг 2: Запустите скрипт
```bash
cd presentations
python generate_powerpoint.py
```

#### Результат:
- ✅ Создастся файл `TD_RUSStankoSbyt_Prezentacija_Zvezdochka.pptx`
- ✅ Профессиональное оформление (темный фон, голубые акценты)
- ✅ Первые 3 слайда как демонстрация структуры

### ⚠️ Важно:
Скрипт создает демонстрационную версию с 3 слайдами (титульный, о компании, ключевые показатели).

Для полной презентации на 16 слайдов рекомендую:
1. Запустить скрипт для получения базовой структуры
2. Открыть созданный PPTX файл
3. Добавить остальные слайды вручную, копируя содержимое из Markdown

## 🎯 Альтернатива: Быстрый ручной метод (15 минут)

Если у вас нет Python:

### В VS Code можно:
1. Открыть файл `Prezentacija_Zvezdochka.md`
2. Установить расширение "Markdown Preview Enhanced"
3. Экспортировать в PDF через меню предпросмотра
4. Открыть PDF в PowerPoint (Файл → Открыть → выбрать PDF)

### Или классический способ:
1. Откройте PowerPoint
2. Создайте темную тему
3. Копируйте содержимое каждого слайда из Markdown
4. Форматируйте вручную (20-30 минут на все слайды)
